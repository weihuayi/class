# Python 数据分析课件

## 内容设计的基本原则

总体目标：帮助学生掌握一系列 Python 数据分析用到的基础编程环境和工具

* 强化各种工具的基础数据结构模型和概念
* 强化基础操作
* 课后大量练习


## 主要内容

1. Python 编程环境搭建 (2 节)
1. IPython 基础 (2 节)
1. Python 语法基础 (4 节)
1. Numpy 基础 (6 节)
1. Pandas 数据处理 (6 节)
1. Matplotlib 数据可视化 (4 节)
1. 机器学习基础 (8 节)
